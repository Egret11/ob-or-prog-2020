Τλοποίθςθ βαςικϊν κλάςεων και ςυναρτιςεων ςε C++
Το δεφτερο παραδοτζο τθσ εργαςίασ απαιτεί τθ δθμιουργία των βαςικότερων κλάςεων του παιχνιδιοφ του Tichu. Συγκεκριμζνα, κα υλοποιθκοφν (τμθματικά) οι παρακάτω κλάςεισ:
1. Κλάςθ Card (card.h και card.cpp)
Η κλάςθ αυτι δθμιουργεί τα αντικείμενα των καρτϊν (φφλλων). Θα πρζπει να περιλαμβάνει μεταβλθτζσ για το είδοσ (suit) και τθν τιμι (value) τθσ κάκε κάρτασ. Επίςθσ κα πρζπει να περιλαμβάνει ςυναρτιςεισ get και set για μεταβλθτζσ τθσ κλάςθσ. Τζλοσ, κα πρζπει να ζχει ςυναρτιςεισ για τον ζλεγχο ιςότθτασ με άλλεσ κάρτεσ, για το αν θ κάρτα μπορεί να ςυμμετζχει ςε ςυνδυαςμοφσ και για το αν μπορεί να ςυμμετζχει ςε βόμβεσ.
2. Κλάςθ Combination (combination.h και combination.cpp)
Η κλάςθ αυτι δθμιουργεί τα αντικείμενα των ςυνδυαςμϊν του παιχνιδιοφ. Θα πρζπει να περιλαμβάνει μεταβλθτζσ για τισ κάρτεσ του ςυνδυαςμοφ. Επίςθσ κα πρζπει να περιλαμβάνει ςυναρτιςεισ για τθν επιςτροφι μιασ κάρτασ του ςυνδυαςμοφ, για τθν επιςτροφι τθσ τιμισ του ςυνδυαςμοφ (που είναι θ τιμι μιασ οποιαδιποτε κάρτασ του) και τον αρικμό των καρτϊν του. Τζλοσ, κα πρζπει να ελζγχεται με ςυνάρτθςθ αν ο ςυνδυαςμόσ μπορεί να παιχτεί μετά από κάποιον άλλο ςυνδυαςμό.
3. Κλάςθ Player (διλωςθ ςτο player.h και υλοποίθςθ ςτα player.cpp, playercombinations.cpp)
Η κλάςθ αυτι δθμιουργεί τα αντικείμενα/ςτιγμιότυπα των παικτϊν. Θα πρζπει να περιλαμβάνει τισ μεταβλθτζσ για το id, το όνομα, τθν κατάςταςθ (status) του παίκτθ, κακϊσ επίςθσ το χζρι (hand) και τθ ςτοίβα με τα φφλλα που ζχει μαηζψει. Επίςθσ κα πρζπει να περιλαμβάνει ςυναρτιςεισ get και set για μεταβλθτζσ τθσ κλάςθσ. Τζλοσ, κα πρζπει να περιλαμβάνει ςυναρτιςεισ για τον ζλεγχο του αν το χζρι του παίκτθ ζχει φφλλα, για το αν ο παίκτθσ ζχει το Mah Jong, κακϊσ και ςυναρτιςεισ για τθν απόφαςθ του παίκτθ να παίξει και για τθν επιλογι του ποιο ςυνδυαςμό κα παίξει.
Σθμείωςθ: Κάποιεσ από τισ μεκόδουσ τθσ κλάςθσ Player ζχουν ιδθ υλοποιθκεί ςτο αρχείο playercombinations.cpp. Εςείσ κα πρζπει να υλοποιιςετε τισ υπόλοιπεσ ςτο αρχείο player.cpp
4. Κλάςθ Hand (διλωςθ ςτο hand.h και υλοποίθςθ ςτα hand.cpp, handcombinations.cpp)
Η κλάςθ αυτι δθμιουργεί τα αντικείμενα/ςτιγμιότυπα των χεριϊν των παικτϊν. Θα πρζπει να περιλαμβάνει τισ μεταβλθτζσ για τισ κάρτεσ που ζχει ςτο χζρι του ο παίκτθσ (και τον αρικμό τουσ), κακϊσ επίςθσ και τουσ ςυνδυαςμοφσ που δθμιουργεί (και τον αρικμό τουσ). Περιλαμβάνει επίςθσ ςυναρτιςεισ για τθν αρχικοποίθςθ του χεριοφ, τθν προςκικθ/αφαίρεςθ καρτϊν, τθν επιςτροφι του πλικουσ των καρτϊν, κ.α. Επίςθσ, περιλαμβάνονται ςυναρτιςεισ για τθν επιςτροφι των ςυνδυαςμϊν και του πλικουσ τουσ. Οι ςυναρτιςεισ αυτζσ είναι υλοποιθμζνεσ, ςυνεπϊσ θ μόνθ ςυνάρτθςθ που καλείςτε να υλοποιιςετε είναι αυτι τθσ καταςκευισ των ςυνδυαςμϊν.
Σθμείωςθ: Όλεσ ςχεδόν οι μζκοδοι τθσ κλάςθσ Hand ζχουν ιδθ υλοποιθκεί ςτο αρχείο hand.cpp. Εςείσ κα πρζπει να υλοποιιςετε μόνο τθ μζκοδο createAllCombinations ςτο αρχείο handcombinations.cpp
Όλεσ οι παραπάνω κλάςεισ κα πρζπει να ζχουν και ςυναρτιςεισ αρχικϊν ςυνκθκϊν. Κατά τθ δθμιουργία του παίκτθ, το status του κα πρζπει να είναι "HASNTPLAYED" και το μζγεκοσ τθσ ςτοίβασ του πρζπει να είναι 0. Επίςθσ, ο παίκτθσ με id 0 κα πρζπει να λζγεται "Player 1", ενϊ ο παίκτθσ με id 1 κα πρζπει να λζγεται "Player 2", κ.ο.κ. Για τθν κάρτα το είδοσ τθσ πρζπει αρχικά να είναι το κενό string και θ τιμι τθσ πρζπει αρχικά να είναι ίςθ με το 0. Τζλοσ, το πλικοσ των καρτϊν ςτο combination κα πρζπει να αρχικοποιείται ςτο 0.
Επιπλζον, κα πρζπει να υλοποιιςετε τισ παρακάτω ςυναρτιςεισ ςτο αρχείο helpers.cpp:
- Συνάρτθςθ numberOfPlayersThatHaveNoCardsLeft
Η ςυνάρτθςθ αυτι χρθςιμοποιείται για να ελεγχκεί αν το παιχνίδι ζφταςε ςτο τζλοσ του. Παίρνει ωσ όριςμα ζναν πίνακα 4 κζςεων που περιζχει τουσ παίκτεσ του παιχνιδιοφ. Θα πρζπει να υπολογίηει και να επιςτρζφει τον αρικμό των παικτϊν που δεν ζχουν κακόλου φφλλα ςτο χζρι τουσ.
- Συνάρτθςθ lastThreePlayersPassedOrHaveNoCardsLeft
Η ςυνάρτθςθ αυτι χρθςιμοποιείται για να ελεγχκεί αν ο γφροσ (trick) ζφταςε ςτο τζλοσ του, ϊςτε ο παίκτθσ που είναι θ ςειρά του να μαηζψει τα φφλλα από το τραπζηι. Παίρνει ωσ ορίςματα ζναν πίνακα 4 κζςεων που περιζχει τουσ παίκτεσ του παιχνιδιοφ και τθ ςειρά του παίκτθ που παίηει. Θα πρζπει να ελζγχει αν οι υπόλοιποι παίκτεσ ζχουν πάει πάςο ι δεν ζχουν πλζον φφλλα ςτο ςυγκεκριμζνο γφρο. Επιςτρζφει true αν ιςχφει θ ςυγκεκριμζνθ ςυνκικθ και false αν δεν ιςχφει.
Σθμείωςθ: Ο παίκτθσ που είναι θ ςειρά του να παίξει ορίηεται ωσ players[playerTurn]
